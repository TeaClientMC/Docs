Docker is a great tool for containerizeing things like nginx and and another portainer.

So as of TeaClient We like containerizeing so we have helped you.

We will provide docker compose files for multiple containers

Running our Website, API, Proxy's, Databases, and more.

```yml
version: "3"
services:
  nginx:
    image: nginx
    ports:
      - 8080:80
  # TODO: @SudoVanillia Nginx config.

  docs:
    image: ghcr.io/teaclientmc/docs:latest
    ports:
      - 80:5050
    volumes:
      - /app/teaclient/docs:.

  website:
    image: ghcr.io/teaclientmc/website:latest
    ports:
      - 80:7053
    volumes:
      - /app/teaclient/website:.

  # API Future We are Looking for contribitor to make this! github.com/teaclientmc/api
```
